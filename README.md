#### 字体图标

```
1、iconfont网站生成矢量图标
	https://www.iconfont.cn/

2、icomoon网站根据矢量图标生成字体图标
	https://icomoon.io/app/#/select
```

#### Web字体

```
css3 开始支持

1、导入web字体
	import './assets/fonts/daysOne.css'
	
2、在样式中设置font-family
	.text{
        font-family: 'Days One';
        font-size: 30px;
        color:orange;
    }
```
